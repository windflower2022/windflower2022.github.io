1 内置对象

JavaScript中的对象分为3种: 自定义对象、内置对象、浏览器对象

前两种对象是JS基础内容,属于ECMAScript;第三个浏览器对象属于JS独有的,JS API讲解

内置对象指JS语音自带的一些对象,这些对象供开发者使用,并提供了一些常用的或最基本的而必要的功能(属性和方法)

常用的有：Math, Date, Array,String等

案例: 猜数字游戏

程序随机生成一个1~10之间的数字,并让用户输入一个数字。

1. 如果大于该数字,就提示,数字大了,继续猜;

2. 如果小于该数字,就提示数字小了,继续猜;

3. 如果等于该数字,就提示猜对了,结束程序。

核心思路:

1. 随机生成一个1~10的整数,需要用到Math.random()方法

2. 需要一直猜到正确为止,所以一直循环

3. 用while循环合适更简单

4. 核心算法: 使用if else if 多分支语句来判断大于、小于、等于

function getRandom(min,max) {

return Math.floor(Math.random() * (max - min + 1)) + min;

}

var random = getRandom(1,10);

while(true) { // 死循环

var num = prompt('你来猜？ 输入1~10之间的一个数字');

if (num > random) {

alert('你猜大了');

} else if ( num < random) {

alert('你猜小了');

} else {

alert('你好帅哦,猜对了');

}

}

习题: 要求用户猜1~50间的一个数字 但只有10猜的机会

function getTime() {

var h = time.getHours();

h = h < 10 ? '0' + h : h;

var m = time.getMinutes();

m = m < 10 ? '0' + m : m;

s = s < 10 ? '0' + s : s;

案例：数组去重(重点案例)

有一个数组['c','a','z','a','x','a','x','c','b'],去除数组中重复的元素

核心思路:

1) 目标：把旧数组里面不重复的元素取出来放到新数组中,重复的元素只保留一个

2) 核心算法: 我们遍历旧数组,然后拿着旧数组元素去查询新数组,如果该元素在新数组里面没有出现过,我们就添加,否则不添加。

3) 如何知道钙元素是否存在?利用新数组.indexOf(数组元素)如果返回时-1就说明里面没有

封装一个去重的函数unique

function unique(arr) {

var newArr = [];

for (var i = 0; i < arr.length; i++) {

if (newArr.indexOf(arr[i]) === -1) {

newArr.push(arr[i]);

}

}

return newArr;

}

var demo = unique(['c','a','z','a','x','a','x','c','b']);

console.log(demo);

案例:返回字符位置

判断一个字符串'abcoefoxyozzopp'中出现次数最多的字符,并统计其次数

// o.a = 1

// o.b = 1

// o.c = 1

// o.o = 4

核心算法:利用charAt()遍历这个字符串

1. 把每个字符都存储给对象 如果对象没有该属性 就为1 如果存在了就+1 如上

遍历对象 得到最大值和该字符

var str = 'abcoefoxyozzopp';

var o = {};

for (var i = 0; i < str.length; i++) {

var chars = str.charAt(i); // chars是字符串的每一个字符

if (o[chars]) { // 得到的是属性值

o[chars]++;

} else {

o[chars] = 1;

}

}

console.log(o);

2.遍历对象

var max = 0;

var ch = '';

for (var k in o) {

// k 得到的是属性

// o[k] 得到的是属性值

if (o[k] > max) {

max = o[k];

ch = k;

}

}

console.log(max);

console.log('最多的字符是' + ch);
