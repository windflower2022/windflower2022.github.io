1. 数组的概念

问：之前学习的变量,只能储存一个值,如果我们想存储班级中所有学生的姓名,那么该如何存储呢？

答：可以使用数组(Array),数组可以把一组相关的数据一起存放,并提供方便的访问(获取)方式.

问: 什么是数组?

答: 数组是指一组数据的集合,其中的每个数据被称作元素,在数组中可以存放任意类型的元素.数组是一种将一组数据储存在单个变量名下的优雅方式

// 普通变量一次只能存储一个值

var num = 10;

// 数组一次可以存储多个值

var arr = [1,2,3,4,5];

2. 创建数组 获取数组中的元素

两种方式： 1) 利用关键字new来创建

var arr = new Array(); //创建一个空的数组

2) 利用数组字面量[]创建

var arr = []; //创建了一个空的数组

var arr1 = [1,2,'pink老师',true];

3) 里面的数据一定用逗号分隔

4) 数组里面的数据 比如1,2称为数组元素

5) 获取数组元素,数组名[索引号]

var arr1 = [1,2,'pink老师',true];

索引号 0 1 2 3

console.log(arr1);

console.log(arr1[2]); //pink老师

数组可以通过索引来访问,设置，修改对应的数组元素,我们可以通过"数组名[索引]"的形式来获取数组中元素

这里的访问就是获取的意思

3. 遍历数组

问：数组中的每一项我们怎么取出来？

答: 可以通过"数组名[索引号]"的方式一项项取出来

var arr = ['red','green','blue'];

console.log(arr[0]);

console.log(arr[1]);

console.log(arr[2]);

问：怎么把数组里面的元素全部取出来?

规律:

从代码中我们可以发现,从数组中取出每一个元素时,代码是重复的,有所不一样的是索引值在递增,答案就是 循环

遍历: 就是把数组中的每一个元素从头到尾访问一次 var arr = ['red','green','blue'];

for (var i = 0; i < 3; i++) {

console.log(arr[i]);

}

1) 因为索引号是0开始,所以i必须从0开始 i < 3

2) 输出的时候 arr[i] 计数器当索引号使用

3.1 遍历数组课堂案例

请将['关羽','张飞','马超','赵云','黄忠','刘备','姜维'];数组里面的元素依次打印到控制台

var arr1 = ['关羽','张飞','马超','赵云','黄忠','刘备','姜维'];

for (var i = 0; i < 7; i++) {

console.log(arr1[i]);

}

1) 数组的长度可使用length来计算,是元素个数,不要跟索引号混淆

2) 所以 i < arr1.length , 数组名.length就无需数个数,它动态检测数组元素的个数

3.2 遍历数组课堂案例

求数组[2,6,1,7,4]里面所以元素的和以及平均值。

思路分析

1) 声明一个求和变量sum

2) 遍历这个数组,把里面每个数组元素加到sum里面

3) 用求和变量除以组的长度就可以得到数组的平均值

var sum = 0;

var arr = [2,6,1,7,4];

var average = 0;

for (var i = 0; i < arr.length; i++) {

sum += arr[i]; // 我们加的是数组元素 arr[i] 不是计数器i

average = sum / arr.length;

}

console.log(sum, average); //想要输出多少个变量,用逗号分隔即可

3.3 遍历数组课堂案例

求数组[2,6,1,77,52,25,7]最大值

思路分析

声明一个保存最大元素的变量max

默认最大值可以取数组中的第一个元素

遍历这个数组,把里面的每个数组元素和max相比较

如果这个数组元素大于max,就把这个数组元素存到max里面,否则继续下一轮比较

最后输出这个max

var arr = [2,6,1,77,52,25,7];

var max = arr[0];

for (var i = 1; i < arr.length; i++) {

if (arr[i] > max ) {

max = arr[i];

}

}

console.log('这个数组里面的最大值是：' + max);

3.4 遍历数组课堂案例



要求: 将数组['red','green','blue','pink']转换为字符串,并且用|或其他符号分割输出：'red|green|blue|pink|'

思路分析:

1) 需要一个新变量用于存放转换完的字符串str

2) 遍历原来的数组,分别把里面数据取出来,加到字符串里面

3) 同时在后加上分隔符

var arr = ['red','green','blue','pink'];

var str = '';

var sep = '|';

for (var i = 0; i < arr.lenth; i++) {

str += arr[i] + sep;

}

console.log(str);

4. 新增数组元素

4.1. 修改length长度

var arr = ['red','green','blue'];

console.log(arr.length);

arr.length = 5; //把我们数组的长度修改为5;里面应有5个元素

console.log(arr);

console.log(arr[3]); //undefined

console.log(arr[4]); //undefined 两个元素为空元素

4.2 通过修改索引号可给出确切的值

var arr1 = ['red','green','blue'];

arr1[3] = 'pink';

console.log(arr1);

arr1[4] = 'hotpink';

console.log(arr1);

arr1(0) = 'yellow'; //本身有元素，这样会替换原来的元素

console.log(arr1);

arr1= '有点意思'; //不要直接给数组名赋值 否则里面的数组元素全都没有了

console.log(arr1);

4.3 案例1

新建一个数组,里面存放10个整数(1-10)

思路分析

使用循环来追加

声明一个空数组arr

循环中的计数器i可以作为数组元素存入

由于数组的索引号是从0开始的,因此计数器从0开始更合适,存入的数组元素要+1

var arr = [];

for (var i = 0;;i < 10; i++) {

// arr = i; 是错的 不要直接给数组名赋值 否则以前的元素都没了

arr[i] = i + 1;

}

console.log(arr);

4.4 案例2-筛选数组

要求：将数组[2,0,6,1,77,0,52,0,25,7]中大于等于10的元素选出来,放入新数组。

思路分析

声明一个新数组用于存放新数据newArray

遍历原来的旧数组,找出大于等于10的元素

依次追加给新数组newArray

方法一

var arr = [2,0,6,1,77,0,52,0,25,7];

var arr1 = [];

var j = 0;

for (var i = 0; i < arr.length; i++) {

if (arr[i] >= 10) {

// 新数组应从0开始 依次递增

arr1[j] = arr[i];

j++;

}

}

console.log(arr1);

方法二

var arr = [2,0,6,1,77,0,52,0,25,7];

var arr1 = [];

// 刚开始arr1.length就是0

for (var i = 0; i < arr.length; i++) {

if (arr[i] >= 10) {

// 新数组应从0开始 依次递增

arr1[arr1.length] = arr[i];

}

}

console.log(arr1);

2. 翻转数组

将数组['red','green','blue','pink','purple']的内容反过来存放

核心思路：

把旧数组最后一个元素作为新元素的第一个元素

声明一个新数组newArr

把旧数组索引号第4个取过来(arr.length-1),给新数组索引号第0个元素(newArr.length)

采取递减的方式 i--

var arr = ['red','green','blue','pink','purple'];

var newArr =[];

for (var i = arr.length-1; i >= 0; i--) {

newArr[newArr.length] = arr [i];

}

console.log(newArr);

3. 数组排序(冒泡排序)

// 交换两个变量

var num1 = 10;

var num2 = 20;

var temp;

temp = num1;

num 1 = num2;

console.log(num1,num2);

冒泡排序:是一种算法（通过观察，得出规律），把一系列的数据按照一定的顺序进行排列显示（从大到小或从小到大）

两两比较后，换序

[5,4,3,2,1]变成[1,2,3,4,5]

var arr = [5,4,3,2,1];

for (var i = 0; i <= arr.length - 1; i++) { // 外层循环趟数

for (var j = 0; j <= arr.length - i - 1; j++) { // 里层循环每趟交换次数

// 内部交换两个变量的值 前一个和后一个比较

if (arr[j] > arr [j + 1]) {

var temp = arr[j] ;

arr[j] = arr[j + 1];

arr [j + 1] = temp;

}

    }

}
console.log(arr);
