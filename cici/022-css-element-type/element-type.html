嵌套规则说明：

设置一个元素的display属性只能改变该元素的显示方式，而不能真正的改变html元素的类型。

比如，一个具有display: inline;的块元素还是不允许在它的内部有其他块元素。

在搭建网页结构时，虽然有很多html元素可以使用，但还是要遵循一些嵌套规则。

规则一：块元素可以包含行内元素或某些块元素，但行内元素却不能包含块元素，它只能包含其它的行内元素；

规则二：块元素不能放在p元素里面。 规则三：有几个特殊的块级元素只能包含行内元素，不能包含块元素，这几个特殊的元素是：h1-h6、p、dt。

规则四：块级元素一般与块级远远并列、行内元素与行内元素并列。
