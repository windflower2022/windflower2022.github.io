062 CSS属性选择器


根据属性或属性值来查找元素。基本语法为:

Element[attribute]

Element[attribute="value"]

意思是查找带有该属性的元素，然后添加样式声明。

a[target] {

background-color: yellow;

}

本段意思是带有target属性的a元素，并为a元素添加了背景为黄色的样式声明。

CSS属性的四种属性：

1.attribute:来查找HTML结构中，带有attribute属性的所有元素

2.attribute="value":来查找HTML结构中，带有attribute属性，并且属性值为value的元素

3.attribute~="value":来查找HTML结构中，带有attribute属性，并且在多个属性值中包含value的元素

4.attribute|="value":来查找HTML结构中，带有attribute属性，并且属性值以value或者是value-开头的元素

总结：

简单选择器和组合选择器，可以完成元素测查找。

其他选择器可以更加精准高效的查找元素，从而能提高我们编码速度。

属性选择器，在实际开发的过程中，大部分会应用在表单控件中，因为表单控件可添加的属性比较多。

即假元素选择器，通过它，可以设置元素指定部分的样式。主要用来设置元素内文本的首字母，首行的样式，或是在元素内容之前或之后插入其他内容。

语法：selector为目标元素，::Pseudo-element,就是向目标元素添加伪元素。

selector::pseudo-element {

color: red;

font-size；3px;

}

伪元素选择器共有5种：

::first-letter:用来实现向文本的首个字符添加特殊样式

::first-line:用来实现向文本的首行添加特殊样式

::before:用来实现元素内容之前插入内容

::after:用来实现元素内容之后插入内容

::selection:用来更改选中文本的样式.只支持color（文本颜色）,background（背景）,cursor（鼠标样式）,outline（描边效果）

054 CSS之内联样式


三种插入样式表的方法：内联CSS,内部CSS,外部CSS

内联CSS：内联样式，行内样式，它被用来为一个单一的元素应用一个独特的样式。样式属性添加到相关的元素中。样式属性可以包含任何CSS属性。如在div里添加style属性，值为任何CSS语法。

  

055 CSS之内部样式


内部CSS: 如果一个单一的HTML页面有一个独特的风格，可使用一个内部样式表。定义在head元素里。需告诉浏览器给哪个html元素添加样式，就需要选择器。

CSS选择器用来寻找或选择你想要定义样式的HTML元素的。

1、元素选择器

根据元素名称选择HTML元素,如body,h3,p1

弊端1：只能作用于一个页面，不能实现多个页面共享一个样式

弊端2：样式代码不断增加，需要上下滚屏，很不便

  

056 CSS之外部样式


外部CSS：可改变一个文件，来改变整个网站外观。将CSS代码放在一个独立的以.CSS为后缀名的文件中，使HTML页面结构文件和CSS样式文件完全独立开来。

每个HTML页面都必须在head元素里添加元素,link元素里定义rel(relationship)属性,值为stylesheet,意思是关联一个样式表。后为href,值为CSS文件的路径。



注意：样式名与中括号1空格，每一条样式属性单独一行定义，两组样式定义空一行分隔。

  

057 CSS样式表的优先级


内联，外部，内部样式修饰同一个html元素，内联样式优先级最高。

在head里添加的内部样式和引入的外部样式，后添加或引入的优先级最高。

浏览器默认的样式优先级最低。

  

058 CSS简单选择器


CSS选择器用来寻找或选择想要修饰的HTML元素。比如前面的h1选择器就是元素选择器。CSS选择器分为5类

1.简单选择器：根据名称，ID，类别来选择元素。

2.组合选择器：根据元素间的特定关系来选择元素。

3.伪类选择器：根据某种状态来选择元素。

4.伪元素选择器：为了一个元素的制定部分设置样式。

5.属性选择器：根据一个属性或属性值来选择元素。

1.简单选择器

元素选择器，如果两段中想要对第二段添加其他样式，就需要ID选择器。

ID选择器：是使用一个HTML元素的id属性来选择一个特定的元素。一个元素的id在页面中是唯一的，所以id选择器是用来选择一个唯一元素的。在前+#。故在第二段P里面加上id属性。

如果想要标题和第二段都设置成斜体，就需要在h1,和#para都设置italic。同一个样式声明写了两遍，很麻烦，可使用类选择器实现。

类选择器：可以选择具有特定class属性的HTML元素。需要写一个点字符后面紧跟类别名称。如.italic。在h1后和第二段P后，使用class="",值可以是1个或多个，可以实现多个样式的叠加。也可以在定义类选择器时候，将元素的名称添加到点的前面。这样的选择器选择的元素就更具体了。

大海是作者诗中的核心意象，广阔浩荡，心旷神怡，生机勃勃，是安魂之乡，是搏斗之乡，是理想之乡，是作者作为“海之子”的精神归宿，是他可以找到真正的幸福感的地方。面朝大海，春暖花开，是一种海市蜃楼，然而这是作者所能感受到的一种明丽的幸福感受。这个题目的意蕴张力在于，它将人们现实中的思维定势扯得七零八落，然后万花筒式地重组，用自己强烈的主导意念使它明丽起来。当然，花应该开在炎热的土地上，作者却让他固执地（“我只愿”）开在自己的幸福之海上，开在自己的理想的追求中，现实中的一切（“喂马，劈柴，关心粮食和蔬菜”），只为了养活自己的躯体，以使自己的心能够跳动，使自己的想象能够飞扬。
