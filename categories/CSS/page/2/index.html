【078】CSS边距

Oct 19, 2022 in CSS
margin

边距

边距塌陷

用边距来控制元素边框外的距离，在元素周围创造空间。区块间的间距使之清晰明了整齐有序。通过margin-top/bottom/left/right

值为auto为自动，lenth以px,pt,cm等为单位指定边距；以父元素宽度的百分比指定边距。为简化代码，可在margin属性中指定所有的边距属性

margin塌陷，发生在相邻块垂直边距和嵌套块元素垂直边距邻接的时候。比如元素1的顶部边距10，和元素的底部边距20会被折叠成单一的20，

这个边距等于两个中最大的那个。不会发生在左右，只会发生在顶部底部。如果你的需要是两个容器（元素）的叠加，只需要给一个元素加上就行

【070】CSS边框

Oct 04, 2022 in CSS
border

border-style:dotted/dashed/solid/double/none/hidden

border-width:px/em/thin,medium,thick

border-color:

写table>(tr>td*3)*3 tab键

左上 中上 右上
左中 中 右中
左下 中下 右下

当表格单元格相邻边框产生冲突时,border-style: hidden>solid>...none优先级高到低。

控制表格中单独的一根线,可在border-style中间添加一个表示方向的词,如border-top-style就可设置单个边框线。可设置top,right,bottom,left方向。除了单个属性外，border-style属性值可使用混合的简写方法

1:四个值如dotted solid double dashed;(上，右，下，左)

2:三个值如dotted solid double;(上，左右，下)

3:两个值如dotted solid:（上下，左右）

4:四个值如dotted(四边)

多个选择器修饰同一个元素，优先级的判断：内联>内部>ID选择器>类选择器>元素选择器>通用选择器*

即行内样式>ID选择器>类选择器>元素选择器>通用选择器*,选择器选择范围小，越精确，优先级越高。

样式发生冲突时，权重值高优先考虑。权重值计算见下面

选择器 例子 权重值

1. 通用计算器 * 0

2.元素选择器 h1,div,p 1

3.类选择器 .active,.light 10

4.ID选择器 #box,#btn 100

5.行内选择器

1000

如：

#box p .light{/* 100+1+10=111 */

color= red;

}

    #box .light {/* 100+10=110 */
        color: red;
    }
