\n 066 字体和字号\n

\n

\n font-family:字体。名称：serif(优雅),sans-serif（圆直）如helvetica,monospace（等宽）,cursive（首先）,fantasy（装饰，娱乐）\n

\n

\n 中文字体：微软雅黑，苹方简，其它字体看自己是否安装。在电脑外观-字体，可查看。\n

\n 先定义的字体优先使用，如果没有就使用默认字体

\n 字号：font-size,值为几px,或几em(16px)/百分数,。默认字号为16px。
